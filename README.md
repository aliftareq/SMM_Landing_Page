# SMM Panel – Pixel-Perfect Social Media Marketing Landing Page

SMM Panel is a modern, responsive, and interactive landing page built to showcase a social media marketing platform. The page is designed to be **pixel-perfect** according to Figma designs, with smooth animations, responsive layouts, and a clean, modern UI.

## 🌐 Live Site

- [SMM Panel Live Demo](https://smmpanel-red.vercel.app/)

## 🔑 Key Features

### 👤 User-Facing Features

- **Responsive Hero Section**
  - Gradient background
  - Custom vector artwork
  - Hero image presentation
  - Login form UI

- **Services Section**
  - Social media platform showcase
  - Animated service cards
  - Interactive platform tabs
  - Custom gradient styling

- **About Section**
  - Product/service highlights
  - Feature points
  - Call-to-action section
  - Responsive layout

- **Working Process Timeline**
  - Four-step workflow design
  - Animated cards
  - Timeline connector line
  - Glass reflection effects
  - Smooth entrance animations

- **Interactive Navbar**
  - Floating glass-effect navbar
  - Smooth scrolling navigation
  - Active section highlighting (Scrollspy)
  - Responsive mobile drawer menu

- **Smooth Animations**
  - Framer Motion powered transitions
  - Left/right content reveal animations
  - Card entrance animations
  - Modern UI interactions

- **Fully Responsive Design**
  - Mobile, tablet, and desktop optimized layouts

## 📋 Summary

This project is a **pixel-perfect recreation of an SMM Panel landing page from Figma** using modern frontend technologies.

The main focus of the project is:

- High-quality UI recreation
- Smooth animations
- Responsive design
- Modern component architecture
- Interactive navigation experience

The combination of **Next.js, Tailwind CSS, Framer Motion, and Lucide React** creates a fast, scalable, and visually polished landing page suitable for real-world marketing platforms.
